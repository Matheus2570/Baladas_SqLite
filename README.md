# 🎉 API de Baladas

Uma API simples para gerenciar informações sobre baladas, incluindo nome, cidade, endereço, tipo e data. Feita com **Node.js**, **Express** e **SQLite**.  

---

## 🛠 Tecnologias usadas

- 🟢 Node.js
- ⚡ Express
- 💾 SQLite
- 📦 NPM
